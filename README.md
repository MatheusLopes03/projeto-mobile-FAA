Criar um aplicativo onde tenha uma tela de login. 
Essa tela de login deve conter: 
 

Campo para informar Email 

Campo para informar Senha 

Link para recuperar a Senha 

Botão para entrar. 

Deve mostrar uma mensagem ao clicar no botão entrar, não precisa validar no backend, pode ser uma mensagem qualquer do frontend. 

Crie também uma tela para  cadastrar usuários, essa tela pode ser de acordo com sua criatividade.
